# Customer Segmentation Project

## 📌 Project Overview
This project analyzes customer data to group customers based on behavior.
📊 Customer Segmentation Analysis
📌 Project Overview
This project focuses on Customer Segmentation using demographic and behavioral data to group customers into meaningful clusters. The aim is to understand customer behavior and identify distinct groups based on attributes such as age, gender, education, profession, work experience, and spending patterns.
By analyzing these patterns, businesses can make better decisions for targeted marketing, customer retention, and personalized services.

🎯 Objectives
Understand customer demographics and behavior
Identify patterns in spending habits
Segment customers into meaningful groups
Provide business insights for marketing strategies
Improve decision-making using data-driven analysis

📁 Dataset Features
The dataset contains the following columns:
ID – Unique customer identifier
Gender – Male or Female
Ever Married – Marital status
Age – Customer age
Graduated – Education status
Profession – Customer occupation
Work Experience – Number of years worked
Spending Score – Score assigned based on spending behavior
Segmentation – Existing customer segment label

🧠 Methodology
Data Cleaning and Preprocessing
Exploratory Data Analysis (EDA)
Feature Understanding and Visualization

📊 Key Insights
Identification of high, medium, and low spending customers
Relationship between age and spending behavior
Impact of profession and education on spending patterns
Differences in spending habits between married and single customers
Discovery of distinct customer groups for targeted marketing
Average Age and Average spending score

📌 Results
The analysis successfully grouped customers into meaningful segments based on their behavioral and demographic patterns. These segments can be used by businesses to:
Target high-value customers
Improve customer retention strategies
Design personalized marketing campaigns
Optimize product recommendations
