# excel-sales-data-analysis
A portfolio of data analysis projects using Excel, SQL, and Python including data cleaning, visualization, and insights.

## 📌 Project Overview
This project analyzes Fuel Expenses to identify trends, highest spending Date, and lowest spending Date.

## 🛠 Tools Used
- Excel
- Google spread sheet
- SQL
- Python

## 📊 Key Insights
- Total Amount Spent
- Total litres purchased
- Highest Spending Date
- Lowest spending Date

## 📁 Files Included
- Raw dataset
- Cleaned dataset
  

## 🎯 Goal
To practice data analysis and present insights in a clear and visual way.
