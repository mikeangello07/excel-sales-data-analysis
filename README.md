A portfolio of data analysis projects using Excel, SQL, and Python including data cleaning, visualization, and insights.

This project analyzes sales data to identify trends, top-performing products, and revenue insights.

## 🛠 Tools Used
- Excel
- SQL
- Power BI

## 📊 Key Insights
- Top-selling product category
- Monthly revenue trends
- Customer purchase patterns

## 📁 Files Included
- Raw dataset
- Cleaned dataset
- Dashboard screenshots

## 🎯 Goal
To practice data analysis and present insights in a clear and visual way.
