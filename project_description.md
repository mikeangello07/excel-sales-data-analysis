# Excel Sales Data Analysis

This project analyzes sales data using Excel and Python.

## Tools Used
- Microsoft Excel
- Python
- Pandas
- Matplotlib

## Project Description
The goal of this project is to analyze sales data to discover insights about product performance, regional sales, and revenue trends.

## Tasks Performed
- Data cleaning
- Data analysis
- Sales trend visualization
- Regional performance analysis

## Key Insights
- Identified the region with highest sales
- Found top selling products
- Analyzed monthly sales trends

- sales_data.xlsx
